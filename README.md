# 校園訂餐系統

## 項目概述
為十所學校設計的訂餐網站，學生需先選擇學校，再填寫姓名、班別、學號登入，然後選擇當日餐點。管理員可輕鬆修改餐單。

## 功能特點

### 學生端
1. **選擇學校** - 十所學校供選擇：
   - 83
   - 世德
   - 亨澤
   - 吳祥川
   - 呂明才
   - 李兆基
   - 周演森
   - 梁植偉
   - 瑪中
   - 崇真

2. **登入系統** - 填寫姓名、班別（1A-6D）、學號

3. **選擇餐點** - 每日五款餐點選擇，包含：
   - 招牌叉燒飯 ($45)
   - 宮保雞丁飯 ($42)
   - 日式咖喱豬扒飯 ($48)
   - 素食豆腐飯 ($38)
   - 黑椒牛柳飯 ($52)

### 管理員後台
- **編輯餐單** - 修改餐點名稱、價格、描述、圖標
- **訂單統計** - 查看今日訂單數量、總收入、最受歡迎餐點
- **訂單列表** - 查看所有訂單詳情

## 使用方法

### 學生訂餐
1. 打開 `index.html`
2. 選擇所屬學校
3. 填寫個人資料登入
4. 選擇喜愛的餐點
5. 確認訂購

### 管理員操作
1. 點擊右下角齒輪圖標 ⚙️
2. 輸入密碼：`admin123`
3. 在「編輯餐單」標籤修改餐點
4. 點擊「儲存餐單」保存更改
5. 在「訂單統計」查看數據

## 技術說明
- 純 HTML/CSS/JavaScript，無需伺服器
- 使用 localStorage 保存餐單和訂單數據
- 響應式設計，支援手機和電腦
- 漸變動畫背景，現代化 UI 設計
